# User App (Flutter) Dokumen ini mencakup panduan untuk pengembang internal dan ringkasan panduan end user aplikasi. - Untuk peta proyek lengkap, lihat ../README.md - Untuk API backend, lihat ../docs/API.md ## Prasyarat (Dev) - Flutter SDK (channel stable) - Android Studio/Xcode, device/emulator ## Setup (Dev) `bash flutter --version flutter pub get ` ## Jalankan (Dev) `bash # Web (opsional) flutter run -d chrome # Android a) nyalakan emulator atau hubungkan device b) flutter run -d emulator-5554 # contoh # iOS flutter run -d ios ` ## Build (Dev) `bash flutter build apk flutter build appbundle # iOS a) flutter build ios ` ## Environment (Dev) - Base API URL menggunakan --dart-define: `bash flutter run --dart-define=API_BASE_URL=https://api.example.com ` - Tambahkan variabel lain sesuai kebutuhan (mis. SENTRY_DSN, FLAVOR) ## Struktur Folder (Ringkas) ` user/lib/ pages/ home.dart widgets/ services/ models/ ` ## Testing & Linting (Dev) `bash flutter test dart format . dart analyze ` ## Panduan End User Aplikasi User - Instal aplikasi dari Play Store/App Store (atau file APK/IPA yang diberikan) - Buka aplikasi, login menggunakan akun terdaftar - Navigasi: gunakan tab/menu bawah untuk berpindah fitur - Pencarian/Filter: tersedia di halaman tertentu - Notifikasi: pastikan izin notifikasi diaktifkan ### Tips Penggunaan - Pastikan koneksi internet stabil - Jika terjadi force close, buka ulang aplikasi dan pastikan versi terbaru ### Bantuan - Hubungi support melalui menu Bantuan/Profil atau kontak yang disediakan ## Troubleshooting - Gagal build Android → flutter clean && flutter pub get, cek versi Gradle/SDK - API tidak merespons → cek API_BASE_URL, dan konektivitas jaringan - Masalah izin (kamera/lokasi) → pastikan izin diaktifkan di pengaturan OS
